# 確認項目

顧客へPDFを渡す前に確認する項目です。

## 入力確認

- 現場名が入っている
- 工事名が入っている
- 着工・竣工が正しい
- 施工者名が正しい
- 施工者の表示／非表示が希望通りになっている

## 写真確認

- 写真がすべて入っている
- 不要な写真が入っていない
- 写真の向きが正しい
- 写真が見切れすぎていない
- 必要な写真は「全体表示」になっている
- 見た目重視の写真は「枠いっぱい」になっている

## 台帳確認

- No.が順番通りになっている
- 工種が正しい
- 自由入力欄の文字が正しい
- 誤字脱字がない
- 空欄のままで問題ない行か確認した

## PDF確認

- 表紙が1ページで出ている
- 写真ページが1ページ3枚で出ている
- 写真が入っていない枠はPDFに出ていない
- 罫線が欠けていない
- 文字がはみ出していない
- ページ順が正しい
- 保存したPDFを開いて確認した

## 保存確認

- 作業途中で「保存」を押した
- PDF作成前にも「保存」を押した
- 保存したデータを必要に応じて開ける
- 顧客へ渡すPDFとは別に控えを残した
