# 操作手順

## 1. 開く

`index.html` をSafariで開きます。  
ホーム画面に追加して使う場合は、Safariの共有ボタンから「ホーム画面に追加」を選びます。

## 2. 表紙を入力する

「入力」画面で、次の内容を入力します。

- 現場名
- 工事名
- 着工
- 竣工
- 施工者
- 施工者を表紙に表示するか

表紙タイトルは「工事写真帳」固定です。

## 3. 写真を入れる

「写真台帳」画面を開きます。  
写真枠を押して、撮影または写真選択を行います。

下のボタンでも操作できます。

- 撮影
- 選択
- 枠追加
- 保存
- 削除
- PDF

## 4. 文字を入れる

各写真枠に、次の内容を入力します。

- No.
- 工種
- 自由入力 5行

No.を整えたい場合は「No.振り直し」を押します。

## 5. 写真の見え方を選ぶ

写真ごとに、次の表示を選べます。

- 枠いっぱい
- 全体表示

顧客へ提出するPDFの見た目を確認しながら選んでください。

## 6. 途中保存する

途中で作業を止める場合は、必ず「保存」を押してください。  
保存したデータは「入力データ読込」から開けます。

このアプリには下書き保存も入っていますが、写真枚数が多い場合は端末容量の都合で下書き保存できないことがあります。  
大事な写真帳は、必ず「保存」で控えを残してください。

## 7. PDFを作る

「PDF」を押します。  
表示される確認内容を見てから進めます。

確認できる内容は次の通りです。

- 表紙ページ数
- 写真枚数
- 写真ページ数
- 合計ページ数
- 保存候補名

問題なければPDFとして保存します。

## 8. 顧客へ渡す前の確認

PDFを保存したら、必ず開いて確認してください。

- 表紙の文字が正しい
- 写真が切れすぎていない
- 工種が正しい
- No.が抜けていない
- 罫線が欠けていない
- ページ順が正しい
