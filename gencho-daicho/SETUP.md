# iPhone使用手順

## 1. GitHub Pagesで開く

リポジトリの Settings → Pages → Branch を `main` / `/root` に設定します。
公開URLの末尾に `/gencho-daicho/` を付けると現調台帳が開きます。

## 2. iPhoneでホーム画面追加

Safariで現調台帳を開く → 共有ボタン → ホーム画面に追加。
以後はアプリ風に起動できます。

## 3. 現調時の基本操作

1. 現場基本情報を入力
2. 現場条件チェックを入力
3. 面積・延長・個数を追加して測定値を入力
4. 劣化箇所を追加
5. 写真枠から撮影
6. 保存
7. 必要に応じてPDF保存、テキスト出力、JSONバックアップ

## 4. データ保存仕様

- 入力内容と写真はブラウザのローカル保存を使用
- 外部サーバー送信なし
- 写真が多い場合は保存容量に注意
- 重要案件はJSONバックアップとPDF保存を併用

## 5. 実務注意

レーザー測定値、スケール実測値、控除根拠は測定行の備考に残してください。後で見積・内訳書・顧客説明に転用しやすくなります。
